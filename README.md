# Rest Fullstack
A simple REST app with Apollo

## Getting started

### Clone the Repo

```sh
> git clone https://github.com/thiagoleitedev/rest-fullstack.git
```

### Enter your newly-cloned folder

```sh
> cd rest-fullstack
```

### Install dependencies

```sh
> yarn install
```
